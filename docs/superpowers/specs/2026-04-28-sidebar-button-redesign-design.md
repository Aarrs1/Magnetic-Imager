# 侧边栏按键优化

## 概述
优化右侧控制栏按键布局：分组、清晰标签、模式高亮反馈、移除冗余快捷键。

## 布局结构
三组分区 + Max Gain，组间有明显间距和分组标题：

```
控制
──────────────────────────
  运行模式（互斥高亮）
┌──────────┬──────────┐
│ ▶ 实时   │ ■ 停止   │
└──────────┴──────────┘

  高速捕获（点击闪一下）
┌──────────┬──────────┐
│  2 kHz   │  1 kHz   │
├──────────┼──────────┤
│ 500 Hz   │ 250 Hz   │
└──────────┴──────────┘

  工具
┌────────────────────┐
│ 🔍 校准  (持续高亮) │
│ 🗑 清空  (点击闪)   │
│ 📷 截图  (点击闪)   │
└────────────────────┘

  Max Gain（输入控件）
┌────────────────┬───┐
│    660 f       │ ▲ │
│                │ ▼ │
└────────────────┴───┘
```

## 交互反馈
- **运行模式**：两个按钮互斥，当前激活态用柔和的绿色（不刺眼）。实时按钮亮 = 实时模式中；停止按钮亮 = 已停止
- **捕获按钮**：点击瞬间有普适色闪（当前按钮的 press_flash 动画即可），不特殊高亮
- **校准按钮**：校准过程中持续高亮提醒（datastore.calibration_enabled 控制），完成后自动恢复
- **清空/截图**：点击瞬间闪一下
- **空格键**：唯一保留的快捷键，切换实时/停止（暂停/继续）
- **所有字母快捷键移除**

## 颜色规范
- 激活态绿色：柔和、偏低饱和度，与现有 frost 暗色风格协调。建议 `rgba(60, 160, 90, ~0.4)` 级别
- 激活态文本：偏白偏亮以示区别

## 实现要点
- `config.py`：重写 `BUTTON_DEFS`，加入分组标记
- `ui_components.py`：按钮支持 `active` 状态渲染（柔绿色背景），分组标题样式
- `main.py`：移除字母快捷键，空格改为切换实时/停止，传递按钮激活态

## 文件影响
- `config.py` — BUTTON_DEFS 改为带分组的格式；新增分组相关常量
- `ui_components.py` — draw_button() 支持 active 参数；新增 draw_section_header()
- `main.py` — 重构按键处理、按钮渲染逻辑、模式状态反馈
