# Max Gain 自定义控制

## 概述
将硬编码的 MAX_VALUE=660 改为用户可调，在右侧控制栏增加输入控件。

## 布局
```
┌─────────────────────┐
│   Max Gain          │  ← 标签
│ ┌──────────────┬───┐│
│ │    660 f     │ ▲ ││  ← 数值 + 上箭头
│ │              │ ▼ ││  ← 下箭头
│ └──────────────┴───┘│
└─────────────────────┘
```
位于右侧按钮栏底部，所有按钮之后。

## 交互
- 点击 ▲：值 +1（最小 1）
- 点击 ▼：值 -1（最小 1）
- 点击数值区域：进入文本编辑模式
- Enter / 点击外部：解析并生效
- Escape：取消编辑，恢复原值

## 影响范围
- `config.py` — MAX_VALUE 保留为初始默认 660.0
- `ui_components.py` — 新增 draw_gain_input()
- `renderer.py` — 颜色条刻度标签改为动态；draw_color_bar_v() 接收 max_gain 参数
- `main.py` — 追踪 max_gain 变量，传递给渲染函数，处理事件
